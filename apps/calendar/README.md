### 🛠️ How to Run
You can now develop and build the project using standard modern web tools:

1. Start Development Server :
   
   ```
   cd apps/calendar
   npm run dev
   ```
   This will start a local server (usually at http://localhost:5173 ) with hot-reloading.
2. Build for Production :
   
   ```
   npm run build
   ```
   This generates optimized static files in the dist/ folder.
