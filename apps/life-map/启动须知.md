### 问题原因
你在直接双击打开 HTML 文件（地址栏显示 `file:///...`），但你的代码中使用了 `<script type="module">` 引入 `app.js`。
现代浏览器**禁止**通过 `file://` 协议加载 ES 模块，因为这会触发跨域（CORS）安全限制，正如报错信息所示：`Cross origin requests are only supported for protocol schemes: http, https...`。

### 技术细节
为了修复这个问题，我在你的应用目录运行了以下命令来启动标准开发服务器：
```bash
npx http-server . -p 8080 -c-1
```
*(参数 `-c-1` 用于禁用缓存，确保你修改代码后刷新能立即生效)*

如果你不想依赖服务器运行（仅用于当前简单功能），也可以修改 `index.html`，去掉 `type="module"` 属性，但建议保持现状并使用服务器，这样方便后续扩展功能。

### 解决方案
你需要启动一个本地 Web 服务器来运行应用。
